# # TikTok Cubano 🇨🇺🎥

Prototipo red social de una app estilo TikTok para creadores cubanos.

**Características simuladas:**
- Monetización por seguidores:
  - ≥ 100 seguidores → pago base ~$5 USD
  - ≥ 1 millón de seguidores → salario mensual ~$1200 USD
- Retiros simulados a Transfermóvil (MLC / USD / CUP / EUR)
- Registro: teléfono, Google, Facebook, X/Twitter

**Importante:** Solo simulación educativa. **NO maneja dinero real ni integra Transfermóvil de verdad.**

## Tecnologías
- Python + Flask
- SQLAlchemy (SQLite para desarrollo)
- Flask-Login

```bash
pip install -r requirements.txt
